1）在人人百壹平台商家消费获得链积分， 用“链积分+金积分”置换交易平台“KEY”，再用“KEY”购买百壹·金链 2）C2C交易（线下用户与用户之间自由置换）正在开发中

2、百壹·金链全球社区根据市场的买卖量，保证购买用户能够买到百壹·金链，卖出用户能卖出百壹·金链，所以系统会自动调控。

3、当市场持续卖出为0的时候，系统将启动强制账户卖出，强制卖出的比例为：1%—30%/用户。

4、强制账户卖出条件：已获得5倍以上收益的用户。（系统按最高持有数量来排序，例：用户A最高持有100个金链，则该天可以卖出1-30个，如有用户需购买30个，则强制卖出30个，100个金链为该账户最高持仓量，而不是根据剩余的比例）；

5、推荐分享：获得直接推荐用户买入百壹金链交易额的2%奖励。

6、交易时间：每周一至周日10:00—16:30

7、百壹·金链平台收费标准：

1）买入百壹·金链手续费：2% 2）卖出百壹·金链手续费：10%

9、单笔买卖限量：10-2000个

单日卖出限量：1%-100%
