百壹·金链全球社区建设

一、指导思想

百壹·金链项目将严格遵循国家相关政策法律法规，并主动接受政府相关监管，百壹·金链将在国家及地方政府授权的区域落地，并遵守各类监管相关要求，争取将百壹·金链相关交易平台打造成为全球区块链产业的应用示范样板项目。打造金链全球社区是我们最重要的工作之一，是金链项目的根基，也是金链生态蓬勃发展的土壤。

二、建设形式

在全球各地，百壹·金链本地社区以金链用户小组（JL User Group）的形式存在，每个用户小组都有负责人、运营团队。

三、参与对象

全球百壹·金链爱好者、开发者、文档维护者、社区活动组织者和百壹·金链持有者等，

四、主要工作

以志愿者的方式投入社区工作。用户组负责组织、维护和发展本地百壹·金链社区。主要工作包括：推广百壹·金链、区块链理念，人工智能、研讨百壹·金链技术，参与百壹·金链项目开发，文档撰写和翻译，组织本地社区会议推广，促成百壹·金链在社区内的相互流通，协助组织金链官方全球性活动。

五、社区管理奖励

一、初级社区经理

1、晋升条件：

1）、百壹·金链上线前合格的社区负责人自动晋级“初级社区经理”， 2）、初级社区经理长期持有价值1万的金链 3）、新晋初级社区经理团队成员在50人以上，成员用链积分置换金链量达10万CNY及以上 4）、至少举办过一次地方“区块链全球推介会百壹·金链专场”的推广会 5）、每月自行组织1场及以上50人左右的白壹·金链推广会

2、奖励机制：获得团队买入交易额的1%奖励

3、经理职责：促进社区用户的“百壹·金链”相互流通。

二、高级社区经理

1、晋升条件：

1）、由初级社区经理晋升，团队成员在80人及以上，成员用链积分置换金链量达20万CNY及以上 2）、高级社区经理长期持有价值2万的金链 3）、成功举办过一次地方“区块链全球推介会百壹·金链专场”的推广会 4）、已报备金链社区项目组，准备召开第二次推广会 5）、会议人数要求：150人及以上 6）、会议主题“区块链全球推介会百壹·金链专场”推广会。 会议内容有1）、区块链是大风口大趋势 2）、百壹·金链交易规则 3）、人人百壹和百壹·金链APP联合运营介绍 7）、每月自行组织2场及以上100人左右的百壹·金链推广会

2、奖励机制

1）、获得团队买入交易额的2%奖励（包含初级社区经理1%） 2）、开通面对面扫码交易功能：省去在平台挂单买卖流程，直接将买单业绩纳入自己团队，促进业绩增长和奖励增加。

3、经理职责

1）、促进社区用户的“百壹·金链”相互流通 2）、培养优质用户升级“初级社区经理”，数量不限。

六、用户推广奖励

直接推荐奖励：被推荐用户买入交易额2%（只获得一级推荐奖励）
